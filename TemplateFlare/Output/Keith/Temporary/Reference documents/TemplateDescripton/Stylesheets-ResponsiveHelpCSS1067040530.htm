ResponsiveHelp.css

This Stylesheet is used for all the topics in a Responsive Help project.

 

 

Responsive Help projects use more than one Stylesheet. This Stylesheet must be manually applied to any new topics.

 

The ResponsiveHelp.css uses the same styles as the A4-Styles.css but with the following style additions.
