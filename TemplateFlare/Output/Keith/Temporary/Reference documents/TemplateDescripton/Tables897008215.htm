Tables

Table styles are determined by table Stylesheets. The table Stylesheet controls the borders, column, and row styling.

Table Stylesheets are available in: Content > Resources > TableStyles

Available table Stylesheets

Table Stylesheet

Description

Normal.css

Standard table with borders top and bottom.

Alternate coloured rows.

Light borders between rows.

FullWidth.css

Standard table.

No indent.

SideHead.css

Header is on the left side with grey background.

Additional column style, "Grey" available for intra-table headings.

Standard tables should be inserted with the following settings:

AutoFit Behaviour: AutoFit to contents

Align: Default

 

 

To create tables with specific cell colours, you cannot use table styles. The table Stylesheet overwrites the local styles each time the Stylesheet is loaded. Use the Default table style and apply the cell styles (td.[class]) defined in the Stylesheet.

 

 

 

Avoid additional formatting of tables after they are used in a topic and linked to a table Stylesheet. These Stylesheets ensure consistent table styles are used throughout documents. If necessary, create a new table Stylesheet.

 

 

 

For specific cell colours, use the cell styles.
