Create a TOC

The TOC (Table of Contents) defines the topics used in the document and the hierarchy between them.

In the Project Organizer view, right-click the TOCs folder and select Add Table of Contents...

The Add File dialog opens.

In Source select New from template > Master-Template

Select the required TOC template and name the TOC.

For more information, see "Available TOC templates" on page 1

It is recommended that you name the TOC the same as the Target name. If the TOC is to be used across multiple output types, provide a meaningful name.

Click OK.

The TOC is created in the TOCs folder.
