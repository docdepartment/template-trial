Topics

All content for a document is contained in Topics. Topics can contain text, images and tables.

New topics specific to one guide should be saved in a guide-specific folder.

Use the styles to determine the look of the content. Styles should be applied based on the meaning of the content. The look is determined by the Stylesheet applied at build time.

According to DITA standards, topics can be Overviews, Tasks and References.

Overview: "The Product comes as part of the doc-department Product Name - PDF service package..."

Task: "To configure the Product, perform the following steps:"

Reference: "The Product has the following dependencies:"

For more information about writing style, refer to the doc-department Style Guide.
