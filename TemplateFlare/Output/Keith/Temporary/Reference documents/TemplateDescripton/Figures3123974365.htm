Figures

Figures are image files within a topic.

Use the p.fig or p.figindent paragraph styles when inserting images

Use the p.figtitle paragraph styles for figure titles / captions.

Figure titles are placed above the figure in the topic.

Use any standard image format for images (.jpg, .png, .eps...), and ensure the image is correctly sized for the application.

Print: 300dpi | max 12cm wide

Online: 96dpi

 

 

For images used in both online and print documents, use MadCap Capture to specify the size and resolution for each output.

 

For more information, refer to the Scale Images page on the doc-department Wiki.

http://localhost:8110/ddwiki/Main/ScaleImages

 

 

Images are stored in Content > gfx > specialgfx.

 

 

 

If there are a large amount of images for several different output types, create a new sub-folder within specialgfx labelled with the document title.
