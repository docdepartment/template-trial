Dashes and hyphens

These are horizontal lines that separate content:

em dash

The em dash is used to add information or examples to a sentence.

• Use the em dash sparingly. Usually a comma can be used in the same place.

Yes: Several versions — the 11k and later — have this function.

Yes: Several versions, for example the 11k and later, have this function.

en dash

The en dash is used for number ranges. It replaces the phrase "from...to".

• Avoid using the en dash for numbers as it can easily be confused with the minus sign.

No:  Temperature ranges: -200-300C

Yes: Temperature range: -200 to 300C

• Use the en dash with a space for combinations of numbers and letters.

Yes: May 2010 - June 2013

hyphens

• Use hyphens to avoid ambiguity.

For example, using this sentence, there can be two meanings depending on where the hyphen is placed.

"We need a biological waste management system."

Yes: We need a biological-waste management system.

We need a system to manage "biological waste".

Yes: We need a biological waste-management system.

We need a "biological" system to manage waste.

• To indicate negative values in numbers, use the minus sign, not a hyphen.
