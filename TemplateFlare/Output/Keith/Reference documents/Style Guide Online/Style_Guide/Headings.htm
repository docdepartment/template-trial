Headings

A heading is the title of the topic subject.

Readers use headings to preview content, so they should be useful and informative. Headings can be one of the following types:

• Generic: The purpose of the content, for example, Introduction.
• Content: The subject of the content, for example, Camera A.

If a reader knows they are not interested in Camera A, a Content heading allows them to skip this section without reading through it first.

Both Generic and Content headings should follow these principles:

• Headings within a section should be of the same type so that readers can quickly find the information they need.
• Do not have an empty topic - a heading but no content.

Always add introductory text, which should be a description of the section.

• Use sentence case capitalization.

No:  How To Set Up The Camera

Yes: How to set up the camera

• Use the correct case when including product names in headings.
• Do not include punctuation.

No:  How do you set up the camera?

No:  How to set up the system.

Yes: How to set up the camera

• Use imperatives and infinitives.

Yes: Open the camera

No:  Opening the camera

• Headings should be less than 10 words long.
• Headings can use variables.
