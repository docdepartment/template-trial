Common Terminology > Common Words > F

F

far-left, far-right

Avoid far-right or far-left; use leftmost or rightmost instead.

fast-forward

Hyphenated.

field

Do not use field to refer to a text box or an option in a dialog.

file extension

Two words.

file name

Two words.

file name extension

Do not use file name extension.

finished

Use finished instead of done, as in 'when you have finished', not the colloquial 'when you are done'.

focus

Do not use focus as a verb in relation to windows or dialogs.

No:  If the window is focused…

No:  Focus the window to…

Yes: Move the focus to the window…

folder

Use folder to refer to a directory or subdirectory.

Reserve directory for specific references to the structure of the file system in documentation and applications.

following

Use following to introduce artwork, a table, or, in some cases, a list. When used for a list, use a colon immediately before the list (except for designated procedure headings).

Yes: The following table compares the different rates.

Do not use following if you want to say after:

No:  Switch off the system following termination of the procedure.

for example

Use for example instead of e.g. to introduce an example clause. Use commas before and after the example clause, or use parentheses if the meaning is unclear. Only use the abbreviation e.g. in tables or legends.

Yes: You can insert an image (for example, a JPEG) into a document.

format

Use format to refer to the overall layout or pattern of a document.

from | than

The adjective different is usually followed by from.

Yes: The result of the first calculation is different from the result of the second.
