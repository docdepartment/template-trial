Common Terminology > Common Words > A

A

above and below

Avoid above and below. Cross-references and content references do not allow the reader to know where the link goes to.

You can use following for lists.

Yes: Use the following items:

No:  The process is shown below:

access key

An access key is a key on the keyboard that corresponds to an underlined letter on a menu, command, or dialog box option.

activate

Avoid activate. Use an alternative appropriate for the intended meaning.

Yes: Open the main menu.

Yes: Start the system scan.

Yes: Switch to the main screen.

active | current

Use active or open to refer to open and operating windows, applications, documents, files, devices, or portions of the screen (such as an open window or active cell).

Use current to refer to a drive, directory, folder, or other element that does not change by itself.

ad hoc

Avoid ad hoc, use improvised.

► See "Language"
add

Use add to refer to creating something new. Use insert to add to something that already exists.

affect or effect

Double-check the use of these words in text.

• Affect is a verb meaning to influence.
• Effect is usually a noun meaning a result. When you affect something, the result can be an effect. Effect can also be a verb meaning to cause something to happen, but this is less common.

Yes: Late software can adversely affect a schedule.

Yes: One effect of late software is schedule slippage.

Yes: A powercut may effect a system shutdown.

allen key

Use lowercase allen unless at the start of a sentence.

allow

Avoid allow or let in the sense of an application permitting or enabling a user to do something; use you can or another alternative.

No:  The Page Layout tab allows you to…

Yes: You can use the Page Layout tab to…

Yes: Use the Page Layout tab to…

► See "can | may"
alphanumeric

One word.

alternative

Strictly speaking, alternative refers to one of two options. However, current usage allows one or more implied options.

among | between

Use among when the emphasis is on distribution rather than individual relationships.

Use between when comparing two items or when denoting a one-to-one relationship, regardless of the number of items.

and/or

Avoid and/or. Usually or alone carries the meaning, but if it doesn’t, rewrite your sentence.

appears

Avoid appears. It suggests a magical quality (contrary to Microsoft). Use display or open instead.

No:  The Patient ID dialog appears.

Yes: The Patient ID dialog opens.

► See "dialog"
appendix (s), appendices (pl)

Spelling: use appendices for the plural.

application

Use application in preference to program.

arrow keys

Use arrow keys to refer collectively to the Left Arrow, Right Arrow, Up Arrow, and Down Arrow keys on the keyboard. Specify on the numeric keypad if you need to differentiate.

Do not use the following:

No:  direction keys, directional keys, movement keys

► See "Banned words"
as

Use as as a conjunction meaning 'in the same manner'. Do not use as a synonym for because or while in subordinate clauses.

Yes: In Excel, as in Word…

No:  Save your files regularly, as the application is unstable.

Yes: Save your files regularly, because the application is unstable.

available | unavailable

Use available and unavailable only for describing user interface controls and hardware or service availability.
