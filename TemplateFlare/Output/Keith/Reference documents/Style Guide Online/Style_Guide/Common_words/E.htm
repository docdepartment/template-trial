Common Terminology > Common Words > E

E

e.g.

Use e.g. only where space is scarce, for example, in table cells.

No:  Image formats (e.g. JPEG, TIFF, and PNG)…

Yes: Image formats (for example, JPEG, TIFF, and PNG)…

earlier

Avoid earlier in the sense of earlier in this section; instead, use a cross-reference.

Use earlier instead of lower for product version numbers, for example:

Yes: Windows® 7 version 4.0 or earlier.

Ethernet

Uppercase.

either/or

Do not use either/or. Fill out the construction, as in:

Yes: You can either save the file or close the dialog without saving.

e-mail

Hyphenated. Can be used as a noun or a verb, as in to send an e-mail, to send by e-mail or to e-mail.

enable, enabled

Avoid enable. In the sense of enabling a command, use make available. In the sense of telling the user something is possible, use you can instead.

end

Use end as a verb to refer to ending a session on the system or to stopping communications and network connections.

Use quit for applications in general; use shut down for the whole system.

Yes: End your user session by logging off.

end user (n), end-user (adj)

Avoid end user; use user instead.

ensure

Means 'to make certain' or 'guarantee'. Do not confuse with insure or assure.

enter

Do not use enter as a synonym for type.

entry field

Do not use entry field to refer to a text-entry field in a dialog; use text box instead.

► See "Banned words"
error message

An error message is a message that tells the user that there has been a problem with the application.

etc.

Abbreviation for the Latin et cetera. Do not use. Use and so on instead.

► See "Banned words"
execute

Do not use execute. Use run, perform, or start instead.

► See "Banned words"
exit

Do not use exit. Use quit as a transitive verb to mean 'close'.

► See "Banned words"
expand, collapse

A user can expand or collapse a folder to see more or fewer subentries. A plus sign next to a folder indicates that it can be expanded to show more folders; a minus sign indicates that it can be collapsed.
