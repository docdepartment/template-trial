W

want

Use want instead of wish, but do not confuse with need.

Always use the term that is appropriate to the situation. Need implies a requirement or obligation; want implies that the user has a choice of actions.

while

Use while to refer to something occurring in time. Avoid as a synonym for although, because this can be ambiguous.

Yes: While the search is in progress, you can continue…

No:  While you can edit text files in Notepad, Word provides more functionality.

Hyphenated.
