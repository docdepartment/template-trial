Common Terminology > Common Words > C

C

camera

Camera is always lower case unless the title of a dialog or window, or at the start of a sentence.

can | may

Use the verb can to describe actions or tasks that the user or application is able to do.

Yes: You can open the file by clicking...

No:  You may open the file by clicking...

Use may only to express possibility or when the result of an action is unknown or variable, not to imply that the user has permission to do something.

Yes: If you do not save your file, your data may be lost.

cancel the selection

Use cancel the selection instead of deselect or unmark. Use clear for check boxes.

cannot

Use cannot as one word; never use the contraction can't.

carry out

Avoid carry out; use perform instead.

No:  Carry out the startup procedure.

Yes: Perform the startup procedure.

check (n, v)

Do not use check as a verb when referring to a check box in a dialog; instead use select or clear.

check box

Use check box, instead of just box. Box alone is ambiguous.

Use the verbs select and clear with check boxes. Do not use turn on and turn off, mark and unmark, check and uncheck, or select and deselect.

clear

Use clear for check boxes; do not use turn off, unmark, uncheck, or deselect.

Yes: Clear the Mirror margins check box.

click

Always use click to describe a sequence of menu actions in a procedure, for example:

Yes: Click File > Print > Print Preview

Do not repeat the verb click unnecessarily, for example:

No:  Click File, then click Print, then click Print Preview

Use select instead of click when choosing menu items.

Do not use click on or click at.

► See "Banned words"
close

Use close for windows, work areas, documents, and dialogs.

Yes: Close the dialog.

Avoid close for applications and network connections: use quit and end, respectively.

command

Users click commands.

Do not use menu item, choice, or option.

► See "Banned words"
configuration

Use configuration instead of setup, for example "the configuration will be displayed…".

configure

Use in configure instead of set up, for example "to configure the application, click…".

confirmation message

A confirmation message confirms to the user that a command has been completed successfully.

context menu

Use context menu to refer to the menu that opens when the user clicks the right mouse button in certain areas (or contexts), such as in a toolbar region.

context-sensitive (adj)

Hyphenated.

could | might

Avoid could and might, because the past tense is seldom necessary and other uses have connotations of politeness or uncertainty that are irrelevant to documentation.

cross-reference (n, v)

Hyphenated.

Ctrl+C, Ctrl+A

Use this style for the Windows keyboard shortcut operations.

No space between the '+' and other shortcut options.

The shortcut option must be sentence case.

Use the same style for mobile keyboard shortcuts.

No:  CTRL+A, CTRL+E

Yes: Ctrl+C

current

Use active rather than current, except for current drive and current folder and for other elements that do not themselves change.
