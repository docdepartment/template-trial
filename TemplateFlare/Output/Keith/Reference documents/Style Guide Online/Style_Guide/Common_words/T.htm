Common Terminology > Common Words > T

T

tab

A labelled group of options (or other controls) used for many similar kinds of settings. Typically tabs are found on the Site Explorer.

For example:

Yes: On the Tools menu, click Options, and then click the View tab.

task

Task refers to something performed by the user. A procedure is a step-by-step description of a task. Do not use job.

taskbar

The bar that appears by default at the bottom of the Windows desktop. Users click buttons that appear on the taskbar to switch between running applications.

text box

A rectangular box in which the user can type text. If the box already contains text, the user can select the default text or delete it and type new text.

Yes: In the Size box, select 10 or type a new font size.

tick

Do not use tick as a noun to mean tick mark, or as a verb when referring to check boxes.

Use select instead.

timeline

One word.

timestamp

One word.

tip

Use a tip to provide supplementary information about a procedure that isn’t included in the procedure itself.

toolbar

Use toolbar instead of navigation bar or icon bar.

torx wrench

Use lowercase torx unless at the start of a sentence.

troubleshoot (v), troubleshooting (adj)

One word.

turn on, turn off

Do not use to refer to selecting or clearing check boxes. Use select and clear instead.

Do not use to mean turning on and off a device, use switch on and switch off instead.
