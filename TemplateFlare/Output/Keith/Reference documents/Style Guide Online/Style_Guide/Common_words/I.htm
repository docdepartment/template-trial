Common Terminology > Common Words > I

I

i.e.

Latin abbreviation for that is. Use only in tables or where space is scarce. Elsewhere, use that is instead.

icon

Use icon only to describe a graphic representation of an object that a user can select and open.

Avoid using icon for dialog options or options that appear on toolbars, on other areas of a window, or on the navigation bar.

No:  Click the Print icon.

Yes: Click Print on the toolbar.

if | when | whether

To avoid ambiguity, use if for uncertainties or conditionals.

Use when for situations requiring preparation or the passage of time.

Use whether for alternatives.

Yes: If you do not know the direct path, you can search…

Yes: Save the file when you are finished.

Yes: Use the displayed file size to determine whether to compress the file.

in order to

Do not use.

No:  In order to create a…

Yes: To create a…

increment

Use as a noun; avoid using as a verb.

Yes: Insert notes in increments of ten.

No:  Increment the version number.

initialize

Use initialize as a technical term usually referring to preparing the system for use.

Do not use instead of start an application or turn on a computer.

input

Avoid using input as a verb; use type instead.

insert

Use insert to add to something that is already there.

Use add to refer to creating something new.

install

Use install when referring to hardware or software. Do not use set up.

Yes: Install the Control Center front-end application.

No:  Set up the Control Center front-end application.

invalid

Avoid using invalid. Use not valid instead.

its or it's

Its is the possessive form; it's is a contraction meaning 'it is'.

Do not use the contracted version. Always expand the words to it is.
