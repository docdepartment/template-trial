Common Terminology > Common Words > D

D

data | datum

Always use data as a plural word with a plural verb; datum is always singular.

Yes: Data are…

No:  Data is…

Yes: An item of data is…

This usage is in line with current grammar and is recommended by the American Medical Association Manual.

database

One word.

decrement

Means to decrease, usually by one unit.

► See "increment"
default

Do not use as a verb.

No:  If you do not choose a different file, it defaults to the last file opened.

Yes: Type a new patient ID or use the default ID of the previous patient.

delete

Use delete to refer to the Delete command and as an imperative in procedures.

Yes: To delete the current order,…

Yes: Use Delete to remove the order…

Do not use cut or erase as synonyms for delete.

► See "Banned words"
depress

Do not use depress. Instead, use press for the action of pushing down a key.

► See "Banned words"
deselect

Do not use deselect. Instead, use cancel the selection, or, for check boxes, clear.

► See "Banned words"
desire

Do not use desire or desired. Instead, use want or required.

Yes: If you want to delete the patient record…

No:  If you desire to delete the patient record…

Yes: Select the required patient.

No:  Select the desired patient.

► See "Banned words"
dialog

Spelling: dialog, not dialogue.

Dialogs open, they do not appear. The user can close a dialog, they do not quit or exit them.

Do not use dialog box.

► See "Banned words"
directory

Use directory only to refer to the structure of the file system. Use folder to refer to the visual representation or object in the interface.

However, do include directory as a synonym for folder in indexes and search topics.

disable

Avoid disable in user documentation. In the sense of disabling a command, use make unavailable.

Yes: The Collate command is unavailable.

No:  The Collate command is disabled.

display

Use display as a verb, for example "The screen displays the application" or "The New File dialog is displayed".

Do not use appear in this context.

Avoid passive constructions if possible:

Yes: The New File dialog is displayed (acceptable, but not recommended).

In procedures, use constructs like:

Yes: In the New File dialog, specify the file name…

Do not use display as a noun. Use monitor to refer to the visual display unit (CRT, LCD etc.). Use screen to refer to the graphic part of the monitor.

done

Do not use when you are done because this is colloquial. Use when you have finished instead.

► See "Banned words"
do not

Use do not as two words.

Do not use the contracted version don't. Not all languages support the contracted format.

► See "Banned words"
double-click, double-clicking

Use double-click and double-clicking in their hyphenated forms. Do not use double-click on.

Yes: Double-click any file in the list.

No:  Double-click on any file in the list.

download (n, v)

One word. It is acceptable to use download to refer to the process of transferring a copy of a file from a remote computer or server.

drag-and-drop

Use drag-and- drop as a verb and a noun. The action of dragging includes dropping the element in place.

Yes: Drag-and-drop files from the Desktop onto the Recycle Bin.

drop-down list

Do not use drop-down list; instead, use drop-down.
