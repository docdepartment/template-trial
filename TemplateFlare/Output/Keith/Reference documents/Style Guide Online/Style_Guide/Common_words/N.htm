Common Terminology > Common Words > N

N

navigate to

Use in preference to go to.

Yes: Navigate to the Alerts tab.

need

Need is often confused with want.

Need indicates a requirement or obligation; want indicates that the user has a choice of actions.

Yes: If you want to use a laser printer, you need a laser printer driver.

non (prefix)

In general, do not hyphenate words beginning with non, unless it’s necessary to avoid confusion. For example:

Yes: nonalphanumeric, nonnumeric, and nonzero

If non is followed by a proper noun, use a hyphen, as in 'non-English'. If in doubt, check with the OED.

normal, normally

Implies 'in a normal manner'. Do not use normal to mean 'often', 'usual' or 'typical'.

note that

Do not use note that. If required, this should be a notice.
