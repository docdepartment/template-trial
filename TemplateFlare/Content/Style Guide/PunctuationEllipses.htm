Ellipsis

An ellipsis is a row of three dots separated from the text without space (for example, there is more...). It should only be used when describing user interface options or as a gap in a series of elements.

Use a space before and after the ellipsis when used as a break in a series.

Press F1, F2, ... , or F8 to select an option.

Ensure the words and ellipsis are contained within the syntactic tags when using an ellipses for user interface components.

Click on Export Locations...

Do not use an ellipsis to indicate a pause or unknown information.

Do not add a full stop at the end of an ellipsis.
