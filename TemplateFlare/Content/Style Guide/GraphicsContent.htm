Content

Graphics should be clear and unambiguous.

Only use a screenshot if it aids the topic description.

Avoid large screenshots. Instead, use cutaways highlighting the area being described.

Minimize the use of text in an image.

Text in an image can present translation issues.

See "Callouts"

Photographs should be well lit and have a high contrast range.

These should still be clear if they are printed in black and white.

The graphic should not lose its meaning if shown as a grey-scale image.

Do not use red and green to highlight differences between graphical objects.

Instead use object shape to highlight any differences.

If showing a graph (bar chart or line chart), show the data in a 2-dimensional view.

Do not use images from the internet without express permission of the graphic owner.

Do not use images which are specific to one country or culture.

See "Localization"
