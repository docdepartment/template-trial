Clarity

Ambiguous and unnecessary information is frustrating and time-consuming for the reader.

Write simple sentences.

Each sentence should contain one discreet piece of information about one topic.

Sentences should be no more than 25 words long.

Use one action per sentence where possible.

This is roughly one verb per sentence, but constructions like 'Click Save and Close' are also acceptable.

Paragraphs should be no more than 5±2 sentences.

Remove useless words.

Check to make sure that the LED is green before…

When the LED is green…

Do not omit accompanying words that are essential to the meaning.

You can change files using the Template utility.

You can change the files by using the Template utility.

Avoid jargon where simpler phrases will do.

Do not use Latin abbreviations and shortenings, except when space is limited (for example, in a table)

e.g., via, per se

p.a. (per annum)
