Capitalization

Capitalization is used to indicate the start of a new sentence or heading. It is also used to indicate proper nouns, for example, product names.

Refer to the client specific style guide for the correct capitalization of product names.

Words must only be capitalized if it is a proper noun or the name of a user interface component.
