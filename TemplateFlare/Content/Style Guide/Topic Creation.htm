Topic construction

A topic is a single piece of content that covers one subject.

Each topic should be self-contained and cover only one piece of subject matter.

Content within a topic should follow the 5±2 principle:

There should be only 5±2 paragraphs in a topic.

Each paragraph should contain only 5±2 pieces of information.

Topics should be no more than one printed page in length.

Topics should have both a heading and body content.
