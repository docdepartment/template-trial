Tasks

Tasks or procedures are a series of numbered instructions that clearly and logically describe a process.

The title of a task must start with an active word in the imperative.

Receive calls

If there are two or more ways of completing a task, always use the preferred method.

Each procedure step should start with an active word in the imperative.

Click, Select, Enter, etc.

If a procedure contains sub-tasks, consider breaking this into a separate topic.

Alternatively, break down the steps into sub-steps, for example:

Click Export Settings...

Locate your setting options

Click the Advanced button

Select the User Level

Your user level is configured during installation.
