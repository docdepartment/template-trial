Reference topics

Reference topics provide specifications or lists of details for part of the product. This can be information which supports tasks or concepts, or gives different specifications for different product versions.

Reference topics typically consist of a table or list of information. This should be in a format which is easily accessible for the reader.

For more information, see "Tables" on page 1

A Reference topic should contain phrases such as:

<The feature> has the following capabilities:

<The feature> has the following requirements: (connections, software, etc.)
