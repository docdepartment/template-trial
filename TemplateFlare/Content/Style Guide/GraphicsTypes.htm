Types of graphics

Avoid BMP or GIF formats.

The list below describes the most common and preferred types and when they should be used.

JPG

JPG files are raster file types and are best used for photograph type images. This graphic type can have large file size, so consider this before you use JPG file type.

EPS

Encapsulated Postscript (EPS) file type is vector-based graphic definition. This graphic type effectively 'draws' the image inside the document. This format is best suited for PDF and printed output. This file type is not supported for online and web outputs.

PNG

PNG files are raster file types and are best used for online outputs.

SVG

Scalable Vector Graphics (SVG) is a vector-based file type. SVG is an XML-based vector image format for two-dimensional graphics.

SVG files are useful if they contain text as this is easy to translate. Ensure the SVG coding supports correct text positioning.
