Structure of a task

DITA tasks must contain the following sections:

Short description that clearly states the objective of the procedure.

Context describing any prerequisites.

Task in discreet steps.

Optionally, post-requisites or results.

Direct users to labels, not UI elements.

Click on the Delete button

Click Delete

The location of each step should come before the action.

Click Search on the Home page

On the Home page, click Search

Start with the goal, not the method.

Turn the dial to increase the volume

To increase the volume, turn the dial

Steps should be short, in the active voice, easy to follow and written in full sentences.
