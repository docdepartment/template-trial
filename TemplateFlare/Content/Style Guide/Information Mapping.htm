Information Mapping

Information Mapping is one approach to developing and presenting information. It comprises a group of key principles which are described in more detail throughout this .

Documents are made up of Blocks and Maps.

Each Block should contain only 5±2 sections.

Each Map should contain only 5±2 Blocks at each level.

The six principles of information mapping are as follows:

Chunking

Labelling

Relevance

Consistency

Accessible detail

Integrated graphics

Information can be one of six types:

Procedure

Principle

Structure

Process

Fact

Concept

Block Labels can be either generic or content-based.
