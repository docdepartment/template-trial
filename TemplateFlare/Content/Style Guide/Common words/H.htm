H

hard copy (n), hard-copy (adj)

Use two words for the noun; hyphenate the adjective.

 hardware

One word.

 he/she

Do not use. Refer to users directly or use the second person pronoun.

He/she can open…

You can open…

For more information, see "Banned words" on page 1

Help

Always with an initial capital letter. In general, use Online Help rather than Help on its own.

However, terms like direct Help and context-sensitive Help are acceptable.

Do not use direct Online Help or context-sensitive Online Help.

 highlight

Avoid highlight, use select instead.

 hint

Do not use hint as a heading for a type of note; use Tip instead.

For more information, see "Safety notices" on page 1

For more information, see "Banned words" on page 1

hyperlink

One word, not hyper link or hyper-link.

Do not use hyperlink as a verb; use link instead.
