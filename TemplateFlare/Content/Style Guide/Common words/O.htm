O

offline

One word.

Use offline in the sense of not being connected to, or part of, a system.

 on

Use on with these elements:

Menus:

The Open command is on the File menu.

Taskbar, navigation bar, toolbar, ribbon, ruler, and desktop:

Click Orders on the navigation bar.

The screen itself:

The image is displayed on the screen.

Network:

The printer is on the network.

Cloud services:

Contacts are stored on the cloud.

 once

Avoid. This is an ambiguous term. Use after or one time.

 online

One word.

 open

Users open windows, files, documents, and folders. They can click an item to open it.

Do not use open to describe choosing a command, a menu, an icon, an option, or other similar element that does not produce a working file in a window. Describe the item as open, not opened.

 option, option button

Use option to refer to the choices in dialogs.

Use initial capitals for the name of an option, but not for the word option itself.

It is usually unnecessary to refer to the button itself, however, only to the option it controls.

Do not use radio button.

 on-screen

Hypenated.

over

To avoid ambiguity, use over to refer to a position or location above something. For quantities, use more than.

Do not use over to refer to version numbers; use later instead:

Windows® 7 version 4.0 or later.
