S

scrollbar

One word.

 select, selected, selection

Use select to refer to marking text, cells, and similar items that will be subject to a user action, such as copying text. Items marked in this way are called selection or selected.

Use select, rather than highlight, because selecting is a standard procedure, whereas highlighting may cause confusion.

 set up (v), setup (n)

Avoid both forms of set up.

Use install to refer to adding an application to a computer.

Use configure to refer to the initial selection of settings in an application. Use configuration (n) to refer to settings that have been selected.

 shortcut key

Refers to a keyboard key or key combination that invokes a particular command such as Ctrl+N or Ctrl+S.

Use shortcut key in all documentation when you are referring to the interface.

 should

Do not use should because it can be ambiguous. Instead, cast your sentence in one of the following ways, depending on the context:

Use the imperative mood.

Use must to specifically instruct users that they must follow some course of action.

Use a phrase such as 'we recommend (in marketing or safety information)'.

You must register with Microsoft to receive free technical support.

We recommend at least 4GB of RAM to run this application.

Rephrase the instruction to recommend some action or condition. For example:

To save different copies of the document, you should save it under different file names.

To save different copies of the document, use a different file name for each copy.

Quit all applications before you shut down your computer.

For more information, see "Banned words" on page 1

shut down (v), shutdown (adj, n), Shut Down (command)

Two words (verb), one word (adjective, noun).

Do not use as a synonym for turning off the power to a computer; instead, use it to refer to the process of quitting all applications before turning off the computer.

Perform the shutdown procedure…

To shut down the system…

Click Shut Down.

 simply | just

Do not use simply or just. It is generally unnecessary and may sound condescending.

For more information, see "Banned words" on page 1

slash

Do not use slash. Use forward slash or backslash.

 status bar

Use status bar, not status line or message area. Messages are displayed on, not in, the status bar.

 step-by-step

Hyphenated.

A step-by-step description of the process ...

 submenu

Avoid submenu if possible, for example, by referring only to what appears on the screen. Avoid the terms cascading menu, hierarchical menu, and secondary menu.

 subsystem

One word.

 such as

Use a comma before such as or use parentheses for the whole clause, if the meaning is unclear.

You can insert an image, such as a JPEG, into the document.

You can insert an image (such as a JPEG) into the document.

 switch

Acceptable to use as a verb, as in 'switch to another window'. Use instead of activate or toggle.

 switch on | switch off

Use switch off for the manual task of switching off an electrical appliance. Use this term in preference to turn off, power off, power down, and shut down, except where the software uses these terms or where the device has no switch.

Use the non-preferred terms only as alternative search keys in the index.

Use switch on for the manual task of switching on an electrical appliance. Use this term in preference to turn on, power on, power up, and start up, except where the software uses these terms or where the device has no switch.

 system

Be careful when using system, as this is generally client-specific.
