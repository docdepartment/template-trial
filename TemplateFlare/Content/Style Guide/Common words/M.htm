M

menus

Use click to describe a sequence of menu actions in a procedure, for example:

Click File > Print > Print Properties

Menus contain commands. Do not refer to a command as a menu item, a choice, or an option.

 message box

In documentation, use message.

 might | could

Avoid might and could, because the past tense is seldom necessary and other uses have connotations of politeness or uncertainty that are irrelevant to documentation.

 more than | over

Use more than to refer to quantifiable figures and amounts.

Use over to refer to a spatial relationship, position, or non-quantifiable amounts.

There are more than 23 tests in this group.

There are over 23 tests in this group.
