grey, greyed

Do not use grey or greyed to refer to commands or options; use unavailable instead.

If you need to describe the appearance of check boxes that show that some portions of a larger selection are already selected, use shaded.

 group box

Two words. A group box is a standard control used to group a set of options or commands in a dialog. For example, in the Samples tab in Orders, the test buttons are in the Profile and Tests group.

In documentation, refer to a group box as a group.

Use the Print Quality group to select print quality settings
