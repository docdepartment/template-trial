back up (v), backup (adj, n)

Note the difference in spelling, depending on use.

Back up your data twice a day. (v)

You can access the backup data… (adj)

Two words.

Two words.

 because | since

Use because to refer to a reason.

Use since to refer to a passage of time. Avoid using since in the sense of 'because'; it is ambiguous.

Save your files regularly, because the application is unstable.

Save your files regularly, as the application is unstable.

below

Avoid below when referring to content.

For more information, see "above and below" on page 1

bottom left, bottom right

Avoid using bottom left or bottom right; instead use lower left and lower right as nouns.

Use lower-left and lower-right as adjectives (hyphenated).

box

Use box instead of field in a dialog to refer to any text box. Avoid using box for a check box (for which you should use the complete term) or a drop-down.

 button

Avoid using button in most cases. If a button is not a common one, or if it is not obvious, you can use button in the first instance.

Click the Open button.

Click Open.

 built in | built-in

Avoid built-in; use built in with no hyphen.
