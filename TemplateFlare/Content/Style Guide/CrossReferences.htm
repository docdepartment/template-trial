Cross-references

Cross-references are used to refer to relevant information elsewhere in the same document or in another document.

When referring to sections within the current document, use the following style:

For more information, see "Setting up a camera"

Refer to / See "Setting up a camera"

Cross-references must be on their own line in a single paragraph.

If the information in the cross-referenced topic is essential to the current topic, include a summary alongside the cross-reference.

In Flare, the topic title is automatically generated by the cross-reference style.

When referring to documents, use the following style:

For more information, refer to the "Control Center Performance Guide"

Use the full title of the document being referenced.

Do not cross-reference to page numbers or topic titles in other documents.

The cross-reference style for referencing other documents usually has to be written manually, including the quoted title text.
