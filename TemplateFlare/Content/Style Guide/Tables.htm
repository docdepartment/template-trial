Tables

Tables are used for presenting reference information so that the reader can quickly find relevant information and cross-reference data.

Tables are not supported well across multiple output sizes. Therefore keep use to a minimum.

Use the formatting styles defined in the template description.

Use a caption above the table. The caption must clearly describe the table contents.

Cells must not contain long or multiple paragraphs.

Do not nest tables within other tables.

Table cell contents must use correct unit notation.

For more information, see "Units and measurements" on page 1
