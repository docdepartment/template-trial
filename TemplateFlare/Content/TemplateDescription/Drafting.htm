Draft notation

Each output type includes draft notification. The following table lists how the draft Variable is used in each output type:

Use of Draft with each output type

Output type

Draft implementation

HTML5 Help

Part of the footer of each topic.

Online Help

Part of the footer of each topic.

PDF

Each page has a diagonal draft stamp and included in the footer text.

Responsive Help

Part of the footer of each topic.

Word

Part of the footer text.

The draft value is defined in the Target's Variables.

When the document is published, remove the draft text. This will remove the draft stamp from the document pages.

For more information, see "Variables" on page 1
