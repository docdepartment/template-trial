Intended Audience

All documents require a topic for Intended Audience, however this is not a common topic. Create this topic and include in it information about the intended audience of the document. Then link the placeholder in the template TOC to this new topic through the Properties dialog.

Ensure that the correct page styles are applied:

Title: p.prefaceHeading2

Body text: p.preface

Ensure that the correct page layouts and page breaks are applied:

Break type: Chapter break

Page layout: Frontmatter

Page type: Normal

End on left: Disabled
