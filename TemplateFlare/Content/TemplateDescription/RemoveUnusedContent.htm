Remove unused content

After you have create your new Target and related TOC, you can remove the template Targets and content that you do not need in your project.

 

 

Do not remove content if you expect to create more output types or TOCs.

 

The following content can be deleted from the Project:

Content view

Style Guide folder and contents

TemplateDescription folder and contents

Project view

Skins folder

Remove the skins that are not required for your output type

Targets folder:

Remove Reference documents folder and contents

Remove template Targets

TOCs folder:

Remove Reference documents folder and contents

Remove MasterTOC
