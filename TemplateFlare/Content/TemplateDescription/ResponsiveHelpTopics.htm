Responsive Help topics

For a Responsive Help Target the following topics must be included:

Home.htm: This is the default startup topic. When a Responsive Help output starts, the default view shows the Home.htm content. However the content displayed consists of three elements:

Top: A Skin (ResponsiveHelp - Top Navigation.flskn) which defines the company logo and initial navigation menu. If the output is displayed on a mobile device the navigation menu disappears and becomes a navigation fly-out icon.

Middle: This is the Home.htm content. This content contains an embedded slideshow application.

This topic can be used to display a 'Hero' image.

For more information, see "Hero image" on page 1

Bottom: This is defined by a Master Page (HomePage.flmsp)

AboutHelp.htm: A topic that contains information about the project, for example, Date, Doc ID, Intended purpose, and so on.
