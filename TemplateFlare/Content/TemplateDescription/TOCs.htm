TOCs

The TOC (Table of Contents) defines the topics used in the document and the hierarchy between them.

 

 

To ensure that the correct headings are applied, topic titles must use a heading style (h1-h4), and Use TOC depth for heading levels is selected in the Target properties.

 

A number of template TOCs are available.

For more information, see "Available TOC templates" on page 1

Ensure that the correct Page layouts and page breaks are applied to individual TOC entries.

For more information, see "Template Structure" on page 1

There are different requirements for the TOC depending the template being used.

For more information, see "Output requirements" on page 1
